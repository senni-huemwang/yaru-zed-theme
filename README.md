# Yaru Zed theme (WIP)

An Ubuntu Yaru theme for Zed Editor, inspired by Paulo Fino's [Adwaita Zed Theme](https://github.com/somepaulo/adwaita-zed-theme)
Includes both light and dark variants.

![Yaru Zed Theme Dark Mode](Screenshot_Dark_Mode.png)

## Installation

Clone the repository into Zed's locally installed extensions directory
